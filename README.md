# Fish Problems

Problems with your aquarium? look no further than Fish Problems! Fish Problems is a React web application that pulls from two separate back ends because why not?

## Technologies Used

-MongoDB
-Express
-React
-Node
-JavaScript
-Skeleton CSS Framework
-Ruby
-Rails
-PostgreSQL
-Netlify
-Heroku
